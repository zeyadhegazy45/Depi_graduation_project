Stroke Data Dashboard

The Stroke Data Dashboard is an interactive tool that visualizes stroke data patterns and risk factors.

The dashboard is built with Streamlit and provides detailed analysis of demographic trends, risk factors, and model insights.

Launch the Dashboard

To start the dashboard, run the following command in your terminal:

streamlit run streamlit_dashboard.py
Open Dashboard

The dashboard will open at http://localhost:8501 by default.
